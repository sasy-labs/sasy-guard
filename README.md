# SASY Policy Enforcement Demo

[![sasy-guard installs/week](https://img.shields.io/pypi/dw/sasy-guard?label=sasy-guard%20installs&color=2563eb)](https://pypistats.org/packages/sasy-guard)

Policy translation and enforcement for LLM agents — write rules in English, enforce them on every tool call.

> 🛡️ **[sasy-guard](https://sasy-demo.pages.dev/claude-code/):** Claude Code security plugin — set it up in minutes.

**[Documentation](https://sasy-demo.pages.dev)** |
**[Quick Start](https://sasy-demo.pages.dev/quickstart/)**

```bash
make setup        # Install dependencies
make translate    # English → verified Datalog
make demo         # Run 9 agent scenarios
make demo-step    # Interactive walkthrough
make docs         # Local docs server
```
