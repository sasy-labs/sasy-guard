# Sasy Guard

[![sasy-guard installs/week](https://img.shields.io/pypi/dw/sasy-guard?label=sasy-guard%20installs&color=2563eb)](https://pypistats.org/packages/sasy-guard)

Sasy Guard is a policy compiler for agentic systems. It turns authorization and
safety rules into deterministic policies that run outside the agent, evaluated
over the agent's message- and data-dependency graph. Because the engine reasons
about where each action's data came from across a whole session, a policy can
decide on the provenance of an action, not just on how the current step looks.
